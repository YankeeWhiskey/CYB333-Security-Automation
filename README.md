# CYB333 Security Automation Project

## Project Objective

This project is a Password Strength Checker developed in Python. The program evaluates passwords based on common security requirements and provides a strength rating.

## Features

- Password length validation
- Uppercase letter detection
- Lowercase letter detection
- Number detection
- Special character detection
- Password strength classification

## Requirements

- Python 3.x

## How to Run

Open a terminal and run:

python3 password_checker.py

## Example Output

Enter a password: P@ssword123

Password Analysis:
Length: Pass
Uppercase: Pass
Lowercase: Pass
Number: Pass
Special Character: Pass
